Team Project for CS32 Week 08 Section

`waylo.md`: Project description.

`city.py`: A slightly changed version of `city.py` from Chapter 11. It's `__init__` method has been tweaked to be more appropriate for this project. It uses an unchanged `maze.py`.

`compare.py`: A framework for comparing two car configurations using a random rider hail stream.

`consts.py`: A collection of constant definitions useful for this project.

`offline.py`: Data structures and functions for offline setup of Waylo cars.

`online.py`: Data structures and functions for online generation of riders and rider hails.

`simulate.py`: Simulation code taking one car configuration and rider hail stream.
